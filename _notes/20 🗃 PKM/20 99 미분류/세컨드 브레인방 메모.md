개발을 주제로, 예로 들면, 제 경우는(글로 쓰려니 너무 장황해질꺼 같은데)

우선 많은 개발 뉴스들에서 관심 가는 내용을 메모로 작성하면서 연관을 시키고요.

사이드 프로젝트나 토이 프로젝트 또는 내가 필요해서 사용할 무언가를 만들 때 기획 문서도 메모로 작성을 하거든요.

예들들면, TextSynth라는 무료 GPT NeoX 20B에 대한 메모를 작성해두고, 나중에 관련 기술 메모 또는 사이드 프로젝트 메모를 작성할 때 이 기술을 참고할 수 있으면, 연결을 해요.

근데, 이 과정에서 천천히(저는 천천히 하나 하나 이전 메모를 읽어보고 있어요.) 메모들을 연결하고, 다른 응용 방법도 찾아보고 그러고 있어요.

이 과정에서 참고할 CSS 관련 메모들도 연결을 해요. 가령 CSS 그라데이션 만들기 메모를 연결하는 식으로요.
이 과정에서 저는 재학습이 많이 되는거 같아요.

Thin에 대한 메모를 작성한다면, React 앱 메모, 백앤드 서비스, 지연 시간, State 관리, 실시간 API 등의 메모와 연결을 해요.
