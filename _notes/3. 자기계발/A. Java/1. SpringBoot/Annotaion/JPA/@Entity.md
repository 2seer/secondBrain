# Today I Learned
## Created : [[2022-05-19]]
## Content
- 테이블과 링크될 클래스임을 나타낸다
- 기본값으로 클래스의 카멜케이스 이름을 언더스코어 네이밍으로 테이블 이름을 매칭한다.
   ex) MarketingPreference.java -> marketing_preference Table
- Entity 클래스에는 [[@Setter]] 메서드를 만들지 않는다.
- 대신 [[@Builder]]를 통해 제공되는 빌더 클래스를 사용한다.
